実験手順

Ⅰ実験準備

1.デバイスを装着してもらう
2.vscode でwitacceralationを開く
3.loggerプログラムを起動 時刻確認、動作確認、ログ削除
4.wit motion起動　デバイス接続
5.wifi,その他アプリを落とす
6.witmotionのデータ収集テスト データ数の違いがないか確認

---------------------------------------------------------
Ⅱ.各回計測

0.演奏の合図について説明

1.机の上に水平になるように手を置いてもらう
2.loggerで曲名、演奏者数、初期データを選択して開始を押す
3.水平時データ計測(4-5秒)
4.成功の場合は成功を選択し終了を押す

5.演奏者、指揮者の方に演奏の準備をしていただく
6.loggerで曲名、演奏者数、計測データを選択
7.お願いしますといい、開始を押す
8.指揮時データ計測(30~60秒)
9.成功の場合は成功を選択し終了を押す
10.記録データを確認し秒数、データ数を確認する

窮屈になったら休憩して頂いて
